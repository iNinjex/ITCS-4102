Programming Languages Final Project Proposal
Project Title
Comparing REST API Development in Python, JavaScript, and Go
Team Members
Individual Project

1. Project Proposal
The goal of this project is to compare three modern programming languages by implementing the same REST API in each language. The project will demonstrate how different programming languages handle syntax, data types, concurrency, libraries, and API development while solving the same problem.
The three programming languages selected are:
Python
JavaScript (Node.js)
Go
Each implementation will provide the same functionality, including:
Create records
Read records
Update records
Delete records (CRUD)
After implementing the APIs, the project will compare the languages based on:
Code readability
Ease of development
Performance
Type safety
Error handling
Library ecosystem
Development experience
Because Artificial Intelligence is becoming an important part of software development, the project will also briefly discuss how AI tools such as ChatGPT and GitHub Copilot assist developers in writing code for each programming language.

4. Main Tasks
The project will be completed individually.
Task 1
Research each programming language and its web framework.
Task 2
Develop a REST API using FastAPI in Python.
Task 3
Develop the same REST API using Express.js in JavaScript.
Task 4
Develop the same REST API using the Gin framework in Go.
Task 5
Test each API using Postman.
Task 6
Measure and compare:
Code complexity
Number of lines of code
Ease of implementation
Runtime performance
Type safety
Error handling
Task 7
Write a final report discussing the advantages and disadvantages of each language and identify which language is best suited for web API development.

Expected Outcome
By the end of the project, the same application will be implemented in three different programming languages. The comparison will provide a better understanding of programming language design, development tools, libraries, performance, and modern software engineering practices. The project will also demonstrate how AI-assisted programming influences software development in today's industry.
